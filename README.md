# Handwritten Digit Recognition using PyTorch (MNIST)

This project is a deep learning application built using PyTorch to recognize handwritten digits (0–9) from the MNIST dataset.

## 📌 Project Overview

The model is trained on the MNIST dataset and can classify handwritten digit images into 10 categories (0–9).  
A simple web interface is created using Flask to allow users to upload images and get predictions.

## 🧠 Technologies Used

- Python
- PyTorch
- Flask
- HTML / CSS
- MNIST Dataset

## 📊 Model Details

- Framework: PyTorch  
- Dataset: MNIST
